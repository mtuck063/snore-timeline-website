# Snore Timeline - Marketing Website

Official marketing website for **Snore Timeline**, an iOS app that visualizes sleep sounds through an interactive timeline.

## About

Snore Timeline helps users track and understand their nighttime breathing and snoring patterns with complete privacy. All processing happens on-device with no cloud uploads or data collection.
